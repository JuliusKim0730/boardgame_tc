# Workflows 수정 사항 정리 (v4.1 기준)

## 📋 개요
게임 룰북 v4.1과 카드 리스트 v4.1의 변경사항을 반영하여 workflows 파일들을 업데이트해야 합니다.

---

## 🔄 주요 변경 사항 요약

### 1. 초기 자원 변경
- **변경 전**: 초기 자금 2,000TC
- **변경 후**: 초기 자금 **3,000TC**
- **영향 파일**: `2.00_게임세팅_초기화.md`

### 2. 집안일 카드 수익 조정
- **변경 전**: 다양한 수익 범위
- **변경 후**: **1,500TC ~ 2,000TC로 통일**
- **카드 수**: 13장
- **영향 파일**: `3.00_턴루프_카드행동.md`

### 3. 여행 지원 카드 리스킨 (구 투자 카드)
- **변경 전**: "투자" 카드
- **변경 후**: "여행 지원" 카드 (감성 스토리 강화)
- **구성**: 수익 8장 (Y1~Y8), 지출 8장 (Y9~Y16)
- **영향 파일**: `3.00_턴루프_카드행동.md`

### 4. 찬스 카드 추가/변경
- **총 26장** (보상 2장, 지출 5장, 상호작용 6장, 드로우 4장, 특수 3장, 캐치업 5장, 기타 1장)
- **신규 카드**: CH19 "반전의 기회" - 이동 없이 현재 칸 행동 1회 추가
- **2인 전용 금지 카드**: CH11(계획 교환), CH12(모두 내 자리로), CH13(자리 바꾸기)
- **영향 파일**: `3.00_턴루프_카드행동.md`, `5.00_동시성_예외복구.md`

### 5. 일반 계획 카드 세부 조정
- **총 40장** (각 특성별 6장)
- 비용 범위: 3,000TC ~ 9,000TC
- **영향 파일**: `3.00_턴루프_카드행동.md`, `4.00_최종정산_리플레이.md`

### 6. 결심 토큰 규칙 명확화
- **기본 최대 보유량**: 2개
- **주간 회복**: 1~6일차 미사용 시 7일차에 1개 회복
- **영향 파일**: `3.00_턴루프_카드행동.md`

### 7. 14일차 규칙 명확화
- 14일차에도 **"이동 → 행동" 필수 수행**
- 모든 플레이어 턴 완료 후 최종 결산
- **영향 파일**: `3.00_턴루프_카드행동.md`, `4.00_최종정산_리플레이.md`

### 8. 비주류 특성 변환 규칙
- **가중치 1배인 비주류 특성 3점** → **추억 +1**로 변환 가능
- 게임 종료 후 한 번만 수행
- **영향 파일**: `4.00_최종정산_리플레이.md`

### 9. 2인 전용 규칙 추가
- 공동 목표 패널티 면제
- 찬스 칸 강화: 찬스 카드 OR 500TC 선택
- 집안일 칸 첫 방문 보너스: +500TC
- 특정 찬스 카드 비활성화 (CH11, CH12, CH13)
- **영향 파일**: `2.00_게임세팅_초기화.md`, `3.00_턴루프_카드행동.md`, `4.00_최종정산_리플레이.md`

### 🔧 2.00_게임세팅_초기화.md

#### 수정 항목:
1. **초기 자원 지급 수정**
   ```
   변경 전: 초기 자원: 돈 2,000, 위치 1, 계획 1 드로우, 결심 토큰 1
   변경 후: 초기 자원: 돈 3,000TC, 위치 1, 계획 1 드로우, 결심 토큰 1
   ```

2. **덱 종류 명칭 변경**
   ```
   변경 전: 덱 6종 셔플: 계획/무료계획/집안일/투자/찬스/공동계획(+여행지)
   변경 후: 덱 6종 셔플: 계획/무료계획/집안일/여행지원/찬스/공동계획(+여행지)
   ```

3. **2인 전용 규칙 초기화 추가**
   - 2인 플레이 감지 시 특정 찬스 카드(CH11, CH12, CH13) 덱에서 제거
   - 2인 전용 플래그 설정

### 🔧 3.00_턴루프_카드행동.md

#### 수정 항목:
1. **행동 4번 명칭 변경**
   ```
   변경 전: 4 투자: 투자 1 드로우, 손익 반영
   변경 후: 4 여행 지원: 여행 지원 카드 1 드로우, TC 변동 반영
   ```

2. **집안일 카드 수익 범위 업데이트**
   ```
   변경 전: (명시 안됨)
   변경 후: 집안일 카드 수익: 1,500TC ~ 2,000TC (13장)
   ```

3. **찬스 카드 로직 업데이트**
   - CH19 "반전의 기회" 추가: 이동 없이 현재 칸 행동 1회 추가
   - 2인 플레이 시 CH11, CH12, CH13 처리 금지 로직 추가

4. **결심 토큰 규칙 명확화**
   - 최대 보유량 2개 명시
   - 주간 회복 규칙 추가 (1~6일차 미사용 시 7일차 +1개)

5. **14일차 규칙 명시**
   - 14일차에도 "이동 → 행동" 필수 수행
   - 모든 플레이어 턴 완료 후 최종 결산 진입

6. **2인 전용 찬스 칸 규칙**
   - 5번 찬스 칸: 찬스 카드 OR 500TC 선택 가능

7. **2인 전용 집안일 칸 규칙**
   - 3번 집안일 칸 첫 방문 시 +500TC 추가 보너스

8. **강제 이동 예외 규칙 명확화**
   - 찬스 카드 등으로 강제 이동 시 "직전 턴 행동 칸 불가" 규칙 미적용

### 🔧 4.00_최종정산_리플레이.md

#### 수정 항목:
1. **비주류 특성 변환 규칙 추가**
   ```
   추가: 비주류 특성 변환 단계
   - 여행지 가중치 1배인 비주류 특성 3점 → 추억 +1 변환
   - 원하는 만큼 반복 가능
   - 게임 종료 후 한 번만 수행
   ```

2. **점수 계산 순서 업데이트**
   ```
   1. 최종 구매 (Hand의 계획 카드 구매)
   2. 공동 목표 판정
   3. 비주류 특성 변환 (신규)
   4. 최종 점수 계산
   ```

3. **2인 전용 공동 목표 규칙**
   ```
   추가: 2인 플레이 시 공동 목표 실패해도 패널티 없음
   ```

4. **동률 규정 명확화**
   ```
   1순위: 추억 점수 높은 플레이어
   2순위: 남은 TC 많은 플레이어
   최종: 공동 우승
   ```

5. **처리 섹션 업데이트**
   - 공동 목표 성공/실패 시 2인 패널티 예외 규칙 명시
   - 비주류 특성 변환 로직 추가
