## 🚀 백엔드 실행 방법

### 1단계: 백엔드 디렉토리로 이동
```bash
cd backend
```

### 2단계: 의존성 설치 (처음 한 번만)
```bash
npm install
```

### 3단계: 백엔드 실행
```bash
npm run dev
```

예상 출력:
```
🚀 Server running on port 3000
📡 WebSocket ready
🌐 Environment: development
```

## 🐛 문제 해결

### "포트 3000이 이미 사용 중"
다른 프로세스가 포트를 사용 중입니다:

**Windows:**
```bash
netstat -ano | findstr :3000
taskkill /PID [PID번호] /F
```

**또는 .env 파일에서 포트 변경:**
```
PORT=4000
```

### "Cannot find module"
의존성 재설치:
```bash
rm -rf node_modules package-lock.json
npm install
```

### 데이터베이스 연결 실패
`.env` 파일 확인:
```
DB_HOST=aws-1-ap-southeast-2.pooler.supabase.com
DB_PORT=6543
DB_NAME=postgres
DB_USER=postgres.xskaefoqkbwnhrpyptkl
DB_PASSWORD=9orkL1p59FjOnZQd
```
