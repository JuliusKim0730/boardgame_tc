## 🎯 게임 플레이 플로우

### 1단계: 로비
1. 닉네임 입력
2. 방 만들기 또는 방 참여하기

### 2단계: 대기실 (신규!)
1. 방 코드 확인
2. 플레이어 목록 확인
3. 방장이 게임 시작

### 3단계: 게임 진행
1. 턴 시작 알림
2. 이동 (인접 칸 선택)
3. 행동 (1~6번)
4. 찬스 카드 상호작용 (모달)
5. 공동 계획 기여 (모달)
6. 14일 반복

### 4단계: 결과 (신규!)
1. 순위 표시
2. 점수 세부 내역
3. 다시하기 또는 로비로
