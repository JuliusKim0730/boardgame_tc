## 2단계: 기본 게임 플레이 테스트

### 테스트 시나리오 1: 2인 플레이
```
1. 방 생성
   - 닉네임 입력: "플레이어1"
   - 방 코드 확인

2. 플레이어 참가
   - 새 탭에서 방 코드 입력
   - 닉네임 입력: "플레이어2"

3. 게임 시작
   - 호스트가 "게임 시작" 클릭
   - 여행지 카드 배분 확인
   - 무료 계획 카드 2장 확인

4. 턴 진행 (Day 1)
   - 플레이어1 턴
     * 이동: 1번 → 2번 (조사하기)
     * 행동: 2번 (계획 카드 1장 획득)
     * 손패 확인: 3장 (무료 2 + 계획 1)
     * 턴 종료
   
   - 플레이어2 턴
     * 이동: 1번 → 3번 (집안일)
     * 행동: 3번 (집안일 카드 획득, 돈 증가)
     * 돈 확인: 3000 + 1500~2000
     * 턴 종료

5. 날짜 전환 확인
   - Day 2 시작
   - 선플레이어 변경 확인

6. 14일 완료까지 진행
   - 각 날짜 정상 진행 확인
   - 8일차 결심 토큰 회복 확인

7. 최종 구매
   - 손패 카드 선택
   - 여행지 카드 정보 표시 확인
   - 구매 완료

8. 결과 확인
   - 점수 계산 확인
   - 여행지 가중치 적용 확인
   - 순위 결정 확인
```

**예상 소요 시간**: 30분

**체크포인트**:
- [ ] 방 생성 및 참가
- [ ] 게임 시작
- [ ] 이동 시스템 (인접 칸만 이동 가능)
- [ ] 행동 시스템 (각 칸의 행동 정상 작동)
- [ ] 턴 종료 및 다음 플레이어 전환
- [ ] 날짜 전환 (선플레이어 변경)
- [ ] 14일 완료
- [ ] 최종 구매
- [ ] 결과 화면

## 4단계: 특수 시스템 테스트

### 결심 토큰 시스템
```
1. 게임 시작 (토큰 1개)
2. 7일차까지 토큰 사용하지 않음
3. 8일차 시작
4. 토큰 개수 확인: 1개 (변화 없음)

5. 새 게임 시작 (토큰 1개)
6. 1일차에 토큰 사용 (0개)
7. 7일차까지 토큰 사용하지 않음
8. 8일차 시작
9. 토큰 개수 확인: 1개 (0→1 회복)
```

**체크포인트**:
- [ ] 8일차 시작 시 0개 → 1개 회복
- [ ] 결심 토큰 사용 시 추가 행동
- [ ] 직전 행동 칸 반복 불가

**예상 소요 시간**: 15분

---

### 2인 전용 규칙
```
1. 2인 플레이 시작
2. 찬스 칸 이동
3. 선택 모달 확인: "카드" or "500TC"
4. "500TC" 선택 → 돈 증가 확인
5. "카드" 선택 → 찬스 카드 획득 확인

6. CH11, CH12, CH13 획득 시도
7. 사용 불가 메시지 확인

8. 집안일 칸 첫 방문
9. +500TC 보너스 확인
```

**체크포인트**:
- [ ] 찬스 칸 선택 모달
- [ ] 2인 금지 카드 사용 불가
- [ ] 집안일 첫 방문 보너스

**예상 소요 시간**: 10분

### 특성 변환
```
1. 최종 구매 완료
2. 특성 변환 모달 확인
3. 비주류 특성 (가중치 1배) 확인
4. 변환 가능 횟수 확인 (비주류 특성 합계 / 3)
5. 변환 수행 (3점 → 추억 +1)
6. 변환 완료
```

**체크포인트**:
- [ ] 비주류 특성 계산

---

### 점수 계산 및 결과
```
1. 특성 변환 완료
2. 결과 화면 확인
3. 여행지 카드 정보 표시 확인
4. 점수 계산 확인:
   - 맛: base × multiplier
   - 역사: base × multiplier
   - 자연: base × multiplier
   - 문화: base × multiplier
   - 레저: base × multiplier
   - 물: base × multiplier
   - 추억: base (×1)
5. 총점 확인
6. 순위 확인
7. 동률 규정 확인 (1순위: 추억, 2순위: TC)
```

**체크포인트**:
- [ ] 여행지 카드 정보 표시
- [ ] 점수 계산 정확성
- [ ] 순위 결정 정확성
- [ ] 동률 규정 적용

**예상 소요 시간**: 5분
