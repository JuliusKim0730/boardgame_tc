# **열네 밤의 꿈 – 공식 룰북 v4.1 (정식 출판용)**

## **by 팀 에보개**

## **0\. 게임 소개 (Game Overview)**

**열네 밤의 꿈**은 여름 방학 14일 동안 꿈에 그리던 여행을 준비하는 과정을 다룬 전략·감성 보드게임입니다. 플레이어는 제한된 기간 동안 자원을 관리하고, 계획을 수집하며, 예상치 못한 변수에 대응하여 자신만의 가장 행복한 여행을 완성해야 합니다.

| 항목 | 내용 |
| ----- | ----- |
| **인원** | 2–5인 |
| **시간** | 40–60분 |
| **연령** | 10세 이상 |
| **화폐** | TC (Travel Coin, 트래블코인) |

## **1\. 구성물 (Components)**

### **1.1. 카드**

* 여행지 카드 (10장)  
* 무료 계획 카드  
* 일반 계획 카드  
* 집안일 카드  
* 여행 지원 카드  
* 찬스 카드 (캐치업 포함)  
* 공동 목표 카드

### **1.2. 기타 구성물**

* 플레이어 말 (5개)  
* TC(트래블코인) 토큰 (다수)  
* 결심 토큰 (다수)  
* 게임판 (6개의 행동 칸)

*(※ 카드의 종류별 효과와 상세 수치는 별도의 **카드리스트 문서**에 정리되어 있습니다.)*

## **2\. 게임 목표 및 승리 조건 (Objective & Victory)**

### **2.1. 게임 목표**

14일의 준비 기간 동안 \*\*가장 높은 '최종 행복 점수'\*\*를 획득하는 것이 목표입니다.

### **2.2. 최종 행복 점수 계산식**

$$\\text{최종 행복 점수} \= (\\text{특성 점수} \\times \\text{여행지 가중치}) \+ \\text{추억 점수} \+ \\text{공동 목표 보너스}$$

### **2.3. 승리 판정 및 동률 규정**

최종 행복 점수가 가장 높은 플레이어가 승리합니다. 점수가 동률일 경우, 다음 순서로 승자를 결정합니다.

1. **1순위:** **추억 점수**가 높은 플레이어 승리.  
2. **2순위:** 추억 점수도 동률이라면, 남은 \*\*TC(트래블코인)\*\*가 많은 플레이어 승리.  
3. **최종:** 모든 항목이 동률이라면 **공동 우승**으로 처리합니다.

## **3\. 게임 준비 (Game Setup)**

1. **게임판 배치:** 게임판을 테이블 중앙에 놓습니다.  
2. **카드 덱 준비:** 모든 카드 덱을 잘 섞어 뒷면이 보이도록 지정된 위치에 배치합니다.  
3. **말 배치:** 각 플레이어는 자신의 말을 **1번 칸**에 둡니다.  
4. **자원 및 카드 분배:**  
   * 각 플레이어는 **초기 자금 3,000TC**를 받습니다.  
   * 각 플레이어는 **결심 토큰 1개**를 받습니다.  
   * 각 플레이어는 **여행지 카드 1장**을 뽑아 모든 플레이어가 볼 수 있도록 자신의 앞에 공개합니다. (개인의 목표 가중치)  
   * 각 플레이어는 **일반 계획 카드 1장**을 뽑아 손에 듭니다.  
5. **공동 목표 공개:** 공동 목표 카드 덱에서 **1장**을 뽑아 모든 플레이어가 볼 수 있도록 공개합니다.  
6. **선 플레이어 결정:** 임의의 방법으로 선 플레이어를 정하고, 턴은 **시계 방향**으로 진행합니다.

## **4\. 게임 진행 (Playing the Game)**

게임은 총 \*\*14일(14 라운드)\*\*로 구성되며, 매일 모든 플레이어가 한 번씩 턴을 가집니다.

### **4.1. 턴 순서**

각 플레이어의 턴은 **① 이동 → ② 행동** 순서로 진행됩니다.

| 순서 | 활동 | 내용 |
| ----- | ----- | ----- |
| **①** | **이동 (필수)** | **6\. 이동 규칙**에 따라 연결된 칸 중 하나로 반드시 말을 이동시킵니다. |
| **②** | **행동 (필수)** | 도착한 칸의 행동을 1회 수행합니다. |
| **③** | **추가 행동 (선택)** | **결심 토큰** 사용 또는 특정 **카드 효과**로 이동 없이 행동 1회를 추가로 할 수 있습니다. (4.2. 추가 행동 규칙 참고) |

### **4.2. 추가 행동 규칙**

결심 토큰이나 카드 효과를 사용해 행동을 추가할 경우:

* **이동 없이** 행동만 수행합니다.  
* **단, 바로 직전에 행동한 칸은 반복할 수 없습니다.** (예: 2번 조사하기 후 곧바로 2번 조사하기는 불가)

### **4.3. 14일차 규칙**

마지막 날(14일차)에도 **평소와 동일하게** 모든 플레이어는 \*\*"이동 → 행동"\*\*을 반드시 수행해야 합니다. 모든 플레이어의 턴이 완료되면, 즉시 **9\. 최종 결산 단계**로 넘어갑니다.

## **5\. 행동 칸 설명 (Action Spaces 1\~6)**

| 번호 | 이름 | 설명 |
| ----- | ----- | ----- |
| **1번** | 무료 계획 | **무료 계획 카드 1장**을 획득하고, 카드에 적힌 특성 및 추억 점수를 즉시 적용합니다. |
| **2번** | 조사하기 | **일반 계획 카드 1장**을 획득하여 손에 듭니다. (구매는 14일차 결산 시) |
| **3번** | 집안일 | **집안일 카드 1장**을 획득하고, 카드에 명시된 **일정량의 TC**를 즉시 수익으로 얻습니다. |
| **4번** | 여행 지원 | **여행 지원 카드 1장**을 획득하고, 카드에 명시된 TC 변동 효과를 즉시 적용합니다. |
| **5번** | 찬스 | **찬스 카드 1장**을 획득하고, 카드 효과(변수, 상호작용, 캐치업 등)를 즉시 적용합니다. |
| **6번** | 자유 행동 | 1번부터 5번 칸 중 **원하는 행동 1개**를 선택하여 수행합니다. (선택한 행동에 대해서도 '직전 행동 반복 불가' 규칙 적용) |

## **6\. 이동 및 행동의 핵심 원칙 (Core Movement Principles)**

### **6.1. 기본 이동의 제약**

플레이어가 **자발적으로** 말을 이동시키는 경우에만 다음의 제약이 적용됩니다.

* 매 턴, 반드시 **인접한 (연결된) 칸**으로 이동해야 합니다.  
* **직전 턴에 행동한 칸**으로는 이동할 수 없습니다.

### **6.2. 강제 이동의 예외**

찬스 카드 등 효과에 의해 말이 강제로 이동하는 경우:

* **이미 행동을 마친 플레이어:** 추가 행동 없음.  
* **아직 행동 전인 플레이어:** 강제 이동된 칸에서 **정상적으로 행동 1회**를 수행합니다.  
  * **예외 적용:** 이때는 **"직전 턴 행동 칸 불가" 규칙이 적용되지 않습니다.** (강제 이동은 자발적 이동이 아니기 때문에 제약에서 벗어납니다.)

## **7\. 결심 토큰 (Determination Token)**

* **효과:** 이동 없이 행동 1회를 추가로 수행합니다.  
* **제한:** 바로 직전에 행동한 칸은 반복할 수 없습니다.  
* **기본 최대 보유량:** **2개**입니다.

### **7.1. 주간 회복 규칙**

* **1일차부터 6일차** 동안 결심 토큰을 **1회도 사용하지 않은 플레이어**는  
* **7일차 시작 시**에 결심 토큰 **1개**를 추가로 획득합니다. (최대 보유량을 초과할 수 없습니다.)

## **8\. 공동 목표 (Shared Goal)**

공동 목표 기여는 **14일차 최종 결산 직후**에 한 번에 진행됩니다.

1. **TC 동시 공개:** 각 플레이어는 공동 목표에 기여할 **TC 금액**을 비밀리에 손에 쥐고 동시에 공개합니다.  
2. **지불 및 총합 계산:** 공개된 TC를 지불하고, 모든 기여 금액의 총합을 계산합니다.  
3. **달성 여부 확인:** 총합이 공동 목표 카드에 명시된 목표 수치에 도달했는지 확인합니다.  
4. **성공 시:** 공동 목표 카드에 명시된 **보너스 효과**를 획득합니다.  
5. **실패 시 패널티:**  
   * **3\~5인 플레이:** 기여 금액이 **가장 적은 플레이어**에게 **추억 \-2** 패널티가 적용됩니다.  
   * **2인 플레이:** 패널티를 적용하지 않습니다. (**13\. 2인 전용 규칙** 참조)

## **9\. 최종 결산 (Final Scoring)**

14일차 행동이 모두 끝난 후, 다음의 순서로 최종 행복 점수를 계산합니다.

### **9.1. 일반 계획 카드 구매 (선택)**

* 플레이어는 보유한 일반 계획 카드 중 원하는 카드를 골라 **TC**를 지불하고 구매합니다.  
* **구매한 카드만** 최종 점수(특성, 추억) 계산에 포함됩니다. 구매하지 않은 카드는 효과가 없습니다.

### **9.2. 공동 목표 판정**

* **8\. 공동 목표 규칙**에 따라 기여하고, 성공/실패 여부에 따른 보너스 또는 패널티를 반영합니다.

### **9.3. 비주류 특성 변환 (선택)**

* 플레이어는 자신의 계획 카드에 기록된 특성 중 **여행지 가중치가 1배**인 **비주류 특성 3점**을 **추억 \+1**로 변환할 수 있습니다.  
* 이 과정은 **원하는 만큼 반복** 가능하며, 게임 종료 후 **한 번만** 수행됩니다.

### **9.4. 최종 점수 계산**

* **2.2. 계산식**에 따라 최종 행복 점수를 확정합니다.

## **10\. 카드 더미 소진 규칙 (Deck Depletion)**

1. **재충전:** 특정 카드 덱이 비었을 경우, 해당 종류의 **버린 카드 더미 전체**를 섞어 새로운 덱으로 사용합니다.  
2. **완전 소진:** 버린 카드 더미마저 없다면, 해당 종류의 카드는 **더 이상 뽑거나 획득할 수 없습니다.**

## **11\. 2인 전용 규칙 (2-Player Variant)**

2인 플레이에서는 상호작용의 영향이 극대화되는 점을 고려하여 다음 규칙을 적용합니다.

### **11.1. 공동 목표 패널티 면제**

* 공동 목표 실패 시, 가장 적게 기여한 플레이어에게 **추억 \-2 패널티를 적용하지 않습니다.**

### **11.2. 찬스 칸 강화 (선택 행동)**

2인 플레이 시 **5번 찬스 칸**에 도착하면 다음 중 하나를 선택하여 수행할 수 있습니다.

* **선택 A:** 찬스 카드 1장 획득  
* **선택 B:** **500TC** 즉시 획득

### **11.3. 집안일 칸 첫 방문 보너스**

* 게임 중 \*\*집안일 칸(3번)\*\*을 처음 방문하여 행동을 수행할 때, 집안일 카드 수익 외에 **추가로 \+500TC**를 획득합니다.

### **11.4. 여행지 추천**

* 전략이 겹치지 않고 더 풍부한 경험을 위해, **서로 다른 여행지 카드**를 선택하는 것을 권장합니다.

## **12\. 플레이 요약 (Player Reference)**

| 구분 | 규칙 요약 |
| ----- | ----- |
| **턴 진행** | **이동 → 행동** (필수) → **추가 행동** (선택) |
| **이동 제약** | 전 턴에 행동한 칸으로 **자발적 이동 불가** (강제 이동은 예외) |
| **추가 행동** | 이동 없음, **직전 행동 칸 반복 불가** |
| **결심 토큰** | 기본 최대 2개, 1\~6일차 미사용 시 7일차에 1개 회복 |
| **14일차** | **이동 & 행동 필수 수행** 후 최종 결산으로 이동 |
| **정산** | 일반 계획은 **TC를 지불하여 구매한 것**만 점수 적용 |
| **변환** | 비주류 특성 3점 $\\to$ 추억 \+1 (게임 종료 후 1회) |
| **2인 특칙** | 공동 목표 패널티 없음, 찬스 칸에서 500TC 획득 선택 가능 |
