## 🔍 코드 검색 쿼리

### 찬스 카드만 조회
```sql
SELECT code, name FROM cards 
WHERE type = 'chance' 
ORDER BY code;
```

### 문화 카드만 조회
```sql
SELECT code, name FROM cards 
WHERE type = 'plan' AND code LIKE 'C%' 
ORDER BY code;
```

### 코드 중복 확인
```sql
SELECT code, COUNT(*) as count
FROM cards
GROUP BY code
HAVING COUNT(*) > 1;
-- 결과 없음 = 정상
```
